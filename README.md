## JMS e ActiveMQ: Messageria com Java

## ActiveMQ

Download em <a href="http://activemq.apache.org/download.html">http://activemq.apache.org/download.html</a>

## Iniciando ActiveMQ

Dentro da pasta /bin

```
sh activemq console
```

Acessar link <a href="http://0.0.0.0:8161/">http://0.0.0.0:8161/</a>
